# Prism
![Prism UI](/images/prismheader.png)

---
A lightweight RAW photo editor for Windows, built with WPF and .NET.

## What is Prism?

Prism is a desktop RAW image editor designed for photographers who want fast and free post-production of RAWs and JPEGs. It supports a wide range of RAW formats and provides real-time adjustments with a clean, dark UI.
